# Piłkarzyki - System Rezerwacji Stołów - Backend

## O projekcie

Projekt "Piłkarzyki" to aplikacja webowa stworzona na zaliczenie przedmiotu "Zwinne Zarządzanie Projektami". System umożliwia rezerwację stołu do gry w piłkarzyki, ułatwiając organizację rozgrywek i zarządzanie dostępnością stołu. Niniejsze repozytorium stanowi backend dla aplikacji dostępnej w https://github.com/SzymonZasada/zwinne-pilkarzyki/tree/main
